## Run Locally

Clone the project

```bash
  git clone https://github.com/Yonokid/PyTaiko
```

Go to the project directory

```bash
  cd PyTaiko
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the game

```bash
  python main.py
```
